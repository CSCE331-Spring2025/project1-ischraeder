Database

In Highschool, I was asked by a local business to design a simple database system for managing inventory. The project was never completed, as the client decided to stick with their old Excel based system for unrelated reasons, but by that time the system was able to handle User management and tracking, the addition of new users to the system with join codes, basic inventory management, and a proof of concept was created for reading barcodes with a camera. The frontend was written in C++ using Webassembly and the QT framework with a backend written in Javascript.
